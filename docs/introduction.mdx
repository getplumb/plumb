# What is Plumb?

Plumb is a two-layer AI memory system designed for persistence across conversations. It stores raw conversation history and extracts structured facts with confidence scoring.

Built as a self-hostable MCP server, Plumb runs locally with SQLite and sqlite-vec for vector search. No cloud required. MIT licensed, open source core.

## Who is it for?

OpenClaw users and Claude Code users who want memory that persists between sessions. Your AI remembers project context, user preferences, and learned patterns without re-explaining every time.

Plumb provides structured memory storage via the Model Context Protocol, integrating directly with any MCP-compatible client.
